**Aslzoda's Calculator** — bu Python va CustomTkinter kutubxonasi yordamida yaratilgan, zamonaviy dizayn va yuqori aniqlikka ega ish stoli kalkulyatori. Ushbu loyiha ham oddiy foydalanuvchilar, ham dasturlashni o'rganayotganlar uchun interfeys dizayni namunasi sifatida yaratilgan.

---

## ✨ Xususiyatlari

* **🌑 Modern Dark UI:** Ko'zni charchatmaydigan, Apple uslubidagi qorong'u mavzu.
* **➕ To'liq Arifmetika:** Qo'shish, ayirish, ko'paytirish va bo'lish amallari.
* **📐 Murakkab Hisoblar:**
    * `^` — Darajaga ko'tarish.
    * `√` — Kvadrat ildiz chiqarish.
    * `%` — Foiz hisoblash.
* **🛡 Xavfsizlik:** `eval()` funksiyasi faqat matematik amallar uchun filtrlanadi.
* **📱 Moslanuvchan:** Har qanday ekran o'lchamida vidjetlar sifatini yo'qotmaydi.
